# Порівняння ефективності алгоритмів пошуку підрядка:
Боєра–Мура, Кнута–Морріса–Пратта та Рабіна–Карпа

## 1. Мета роботи

Метою роботи є емпіричне порівняння ефективності трьох класичних алгоритмів пошуку підрядка:

- **Boyer–Moore**
- **Knuth–Morris–Pratt (KMP)**
- **Rabin–Karp**

Порівняння здійснюється на основі двох текстових файлів:

- `article1.txt`
- `article2.txt`

Для кожного тексту алгоритми тестувалися на:

1. **Існуючому підрядку**, який гарантовано зустрічається в тексті  
2. **Вигаданому підрядку**, якого у тексті не існує

Для вимірювання часу виконання використано модуль `timeit`.

## 4. Аналіз результатів

### 4.1. Для обох текстів:

- Алгоритм **Боєра–Мура** стабільно найшвидший:
  - для існуючих підрядків;
  - для вигаданих підрядків;
  - для обох статей.

- **KMP** програє Боєру–Муру за швидкістю у 8–12 разів.

- **Rabin–Karp** найповільніший через накладні витрати на обчислення хешів.
